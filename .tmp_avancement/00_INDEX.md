## Périmètre fonctionnel

ATS (Applicant Tracking System) complet :
- Cycle de vie complet : besoin → offre → sourcing → candidatures → évaluation → décision → offre → intégration
- Multi-canal : site carrière, jobboards (LinkedIn, Indeed, Glassdoor + jobboards africains), réseaux sociaux, cooptation, agences
- Pipeline visualisé en kanban
- Parsing CV automatique PROPH3T (extraction structurée)
- Scoring multi-critères + matching IA
- Entretiens avec calendrier intégré (lien Google Calendar / Outlook)
- Tests intégrés (techniques, psychométriques)
- Conformité RGPD africaine + non-discrimination OHADA
- Marque employeur (site carrière personnalisable)
- Cooptation avec primes
- KPI complets (time-to-hire, cost-per-hire, quality of hire)

## Public utilisateur

- **Recruteur** : opérationnel sourcing, candidatures, entretiens
- **Chargé recrutement** : gestion pipeline, coordination
- **Responsable recrutement** : pilotage, validation, KPI
- **Manager opérationnel** : valide besoin, conduit entretiens, donne avis
- **RRH/DRH** : valide budget recrutement, signe offre finale
- **DAF** : valide budget poste (impact masse salariale)
- **Juriste social** : valide conformité (RGPD, non-discrimination)
